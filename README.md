## **Compilar SCSS**

* Directo con sass
```bash
      $ pwd
      $ path/blog-barbershop
      $ sass --style=expanded --no-source-map assets/scss/main.scss assets/css/main.css
```
* Directo con npm script
```bash
      $ pwd
      $ path/blog-barbershop
      $ npm run sass
```


## **Capitulos**
* SASS.
* Importar Font-Awesome y Google Fonts.
* Usar Flexbox
